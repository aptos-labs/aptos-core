Struct serde_urlencoded::Deserializer

source · [−]
pub struct Deserializer<'de> { /* private fields */ }
Expand description

A deserializer for the application/x-www-form-urlencoded format.

  • Supported top-level outputs are structs, maps and sequences of pairs, with or without a given length.

  • Main deserialize methods defers to deserialize_map.

  • Everything else but deserialize_seq and deserialize_seq_fixed_size defers to deserialize.
