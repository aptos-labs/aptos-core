Expand description

Panics if the first expression is not less than or equal to the second.

Requires that the values implement Debug and PartialOrd.

On failure, panics and prints the values out in a manner similar to prelude’s assert_eq!.

Example

use more_asserts as ma;

ma::assert_le!(4, 4);
ma::assert_le!(4, 5);
ma::assert_le!(4, 5, "With a message");
ma::assert_le!(4, 4, "With a formatted message: {}", "oh no");
