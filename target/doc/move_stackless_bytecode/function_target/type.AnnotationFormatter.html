pub type AnnotationFormatter = dyn Fn(&FunctionTarget<'_>, CodeOffset) -> Option<String>;
Expand description

A function which is called to display the value of an annotation for a given function target at the given code offset. The function is passed the function target and the code offset, and is expected to pick the annotation of its respective type from the function target and for the given code offset. It should return None if there is no relevant annotation.
