# Errors

When making requests to <CompanyName /> API, you may receive an error. The following is a partial list of the possible errors and their corresponding codes:

## Code 4000

- **Code**: 4000
- **Name**: No Accounts
- **Message**: No accounts found.

## Code 4001

- **Code**: 4001
- **Name**: User rejection
- **Message**: The user rejected the request.

## Code 4100

- **Code**: 4100
- **Name**: Unauthorized
- **Message**: The requested method and/or account has not been authorized by the user.
