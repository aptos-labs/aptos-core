pub enum Sleep {
    RealSleep(RealSleep),
    MockSleep(MockSleep),
}
Expand description

A Future that resolves after some time has elapsed (either real or simulated, depending on the parent TimeService).

Sleep is modeled after [tokio::time::Sleep].
